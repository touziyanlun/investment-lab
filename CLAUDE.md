# 投资严论 · The Rigorous Investor

多资产长期价值投资独立研究。

## 投资哲学

遵循 **查理·芒格** 的多元思维模型与 **沃伦·巴菲特** 的价值投资理念：
- 运用多元思维模型（心理学、经济学、工程学、生物学）理解商业本质
- 寻找有深厚护城河的优质企业，以合理价格买入并长期持有（5 年以上）
- 反过来想，总是反过来想 — 每笔投资先假设它会归零，找出最可能的死因
- 聚焦消费/科技/能源/金融赛道，能力圈内深度研究

## 投资框架

- **核心理论**：摩根士丹利超级周期 + 赛道理论（选好赛道 → 选好公司）
- **投资期限**：5 年以上
- **最大回撤容忍**：10-20%
- **资产范围**：美股 / A股 / 港股 / ETF / 加密货币
- **个股上限**：10%
- **行业偏好**：科技、消费、能源、金融

## 分析框架（每只股票必覆盖）

1. **商业模式** — 怎么赚钱？收入结构、客户粘性
2. **护城河** — 为什么对手抢不走生意？护城河在变宽还是变窄？
3. **竞争对手** — 行业格局、份额变化
4. **行业周期** — 处于周期的什么位置？
5. **催化剂与风险** — 未来 1-3 年的关键变量
6. **估值** — 定量 DCF/PE/可比估值，给出合理区间
7. **管理层与治理** — 资本配置能力、诚信记录、股权结构是否对 minority shareholder 友好
8. **逆向验证（Pre-mortem）** — 假设3年后这笔投资腰斩甚至归零，最可能的死因是什么？把这个死因当作真实威胁重新审视，不要用"这次不一样"轻易否定

## 分析纪律（防偏见）

### 硬数据要求
- 收入拆分必须精确到业务线，不得用"约 XX%"模糊替代——如果找不到精确数据，标注"待验证"并给出置信区间
- 现金流质量检查：经营现金流/净利润 比率连续3年趋势，应收账款增速 vs 营收增速，存货结构变化
- 关联方交易/大客户集中度必须量化，客户依赖度 >50% 直接写进结论

### 反向验证
- 每份报告必须搜索并认真对待至少一份负面/做空论点（Citron、浑水、财经媒体批评文章），正面回应而非反驳
- 估值用至少两种方法交叉验证，如果两种方法给出相差 >30% 的结论，必须解释原因
- 机构一致预期（EPS/目标价）必须与自己的独立估算对比，偏差 >20% 要深究

### 组合维度（每份新研究必做）
- 这笔新标的和已有持仓/研究的公司之间，在什么情景下会同时下跌 >30%？
- 给出与组合中现有标的的估计相关系数和逻辑链
- 如果组合已经高度集中在某个赛道（如半导体超过总研究的 50%），在报告开头明确标注

### 追踪与复盘
- 每份报告必须记录：建议买入价、实际买入价和日期、卖出价和日期、最终收益率
- 每季度更新持仓复盘：当时的买入逻辑还成立吗？什么信号会改变判断？
- 退出条件必须提前定义：股价达到什么水平、基本面出现什么变化、或者发现什么更好的机会时会卖出

## 宏观叠加（每季度更新）

- 中美利差与汇率趋势 → 影响跨境资金流动和持仓成本
- 关键关税/出口管制动态 → 直接影响半导体等赛道
- 全球流动性指标（美联储资产负债表、中国社融增速）

## 数据要求

- 每次分析必须全网搜索最新数据，不得假设或使用模型训练数据
- 优先来源：SEC Edgar / 港交所披露易 / 东方财富 / Yahoo Finance / Bloomberg / FRED

## 代码规范

- 所有代码使用中文注释
- 数据处理脚本放 `scripts/`，分析 Notebook 放 `notebooks/`

## Python 环境

- **虚拟环境**：`venv/`（Python 3.9.6）
- **激活命令**：`source venv/bin/activate`
- **依赖文件**：`requirements.txt`
- **核心包**：`yfinance`（美股数据）/ `akshare`（A股数据）/ `pandas` / `numpy` / `matplotlib`

- **预览**：`mkdocs serve` → http://localhost:8000
