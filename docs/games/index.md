## 🐍 贪吃蛇

经典街机贪吃蛇，**手机 / PC 均可游玩**。操控方式：

| 方式 | 说明 |
|------|------|
| 键盘 | 方向键 ↑ ↓ ← → |
| 手机 | 屏幕下方 D-pad 按钮 |
| 滑动 | 在游戏画面上滑动 |
