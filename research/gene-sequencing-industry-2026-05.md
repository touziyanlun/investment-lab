## 一、行业概览

### 1.1 市场规模

| 指标 | 数据 | 来源 |
|------|------|------|
| 全球 NGS 市场（2026） | $138 亿 | MarketsandMarkets |
| 全球 NGS 市场（2031E） | $271 亿 | CAGR 14.5% |
| 中国基因测序市场（2026E） | $42 亿 | 占全球 ~11% |
| 单细胞测序（2026） | $13.8 亿 | CAGR 15% |
| 长读长测序（2026） | $16.7 亿 | 增速最快子赛道 |
| 三代测序市场 | 加速增长 | PacBio + ONT 主导 |

### 1.2 技术路线

```
基因测序技术树
│
├─ 二代测序（NGS，短读长）— 市场主导（>80%份额）
│   ├─ Illumina：边合成边测序（SBS），全球 ~70%
│   ├─ MGI/华大智造：DNA纳米球（DNBSEQ），全球装机量 ~28%
│   └─ Thermo Fisher：离子半导体（Ion Torrent），利基
│
├─ 三代测序（长读长）— 增速最快
│   ├─ PacBio：单分子实时测序（SMRT），HiFi reads
│   └─ Oxford Nanopore：纳米孔测序，便携实时
│
└─ 四代测序（前沿）— 尚需市场验证
    └─ 蛋白纳米孔 / 固态纳米孔 / 单分子实时合成
```

### 1.3 行业关键趋势

| 趋势 | 影响 |
|------|------|
| **「全读长」时代到来** | MGI 率先实现短读长+长读长双平台商业化（CycloneSEQ） |
| **测序成本持续骤降** | 全基因组从 $1000 → $100，推动临床级普及 |
| **AI × 基因组学** | AI 辅助变异检测、多组学整合成为差异化竞争力 |
| **地缘政治重塑格局** | 美国《生物安全法案》限制华大系 → 反而加速 MGI 自主创新 |
| **Illumina 护城河在变窄** | Signify Research：「Illumina's Moat Drying Up」 |

## 二、五大巨头八维逐一拆解

### 2.1 Illumina（因美纳）— 曾经的绝对王者

| 维度 | 评估 |
|------|------|
| **商业模式** | 剃刀-刀片：测序仪（低毛利）+ 试剂耗材（高毛利，占收入 70%+）。全球装机量 ~23,000 台 |
| **护城河** | ★★★★☆（在变窄）。SBS 专利墙 + 全球最大装机基数 + 客户生态锁定（实验室习惯了 Illumina 的工作流）。但：核心专利 2023-2025 年陆续到期，MGI 用更低成本蚕食份额 |
| **竞争对手** | MGI 是最大威胁——装机量年增 28%，价格低 30-50%。PacBio/ONT 在长读长市场夹击 |
| **行业周期** | 成熟期。NGS 市场增速 14.5%，但 Illumina 增速已降至个位数——份额在被蚕食 |
| **催化剂** | NovaSeq X 系列（2023 年推出）生命周期仍在，临床诊断（NIPT/肿瘤）渗透率提升 |
| **估值** | PE ~35x（2026），历史中枢 40-50x。市场给了平庸增速下的溢价，因为「护城河惯性」 |
| **管理层** | 2023 年 CEO 更换后战略摇摆（Grail 收购/剥离折腾）。管理层动荡削弱了竞争力 |
| **Pre-mortem** | **最大死因**：MGI 在核心测序仪性能上追平 Illumina，同时价格低 50%，客户大规模迁移。概率：中等（30-40%），时间窗口 3-5 年 |

**护城河评分：★★★☆（从 5 星降至 3.5 星，且在持续变窄）**

---

### 2.2 MGI/华大智造 (688114) — 唯一的全能挑战者

| 维度 | 评估 |
|------|------|
| **商业模式** | 与 Illumina 相同（剃刀-刀片），但多了「全读长」差异化——唯一同时提供短读长（DNBSEQ）和长读长（CycloneSEQ）的商业化平台 |
| **护城河** | ★★★★（在变宽）。① DNBSEQ 专利自主，不受 Illumina 专利约束；② 全球首个全读长商业化平台；③ 中国本土市场绝对主导（新增装机份额 63.8%）；④ 成本优势 30-50% |
| **竞争对手** | Illumina（正面竞争）、PacBio/ONT（长读长市场） |
| **行业周期** | 加速期。中国基因测序渗透率远低于美国，市场增量空间大 |
| **催化剂** | ① CycloneSEQ 长读长量产（2025 年起）→ 打开长读长市场；② 美国市场恢复（生物安全法案影响边际递减）；③ AI+多组学驱动新需求 |
| **估值** | PS ~6x（2025 营收 27.8 亿），亏损收窄中（-2.22 亿，+63% YoY）。市场给的是「挑战者折价」而非「颠覆者溢价」 |
| **管理层** | 汪建（华大集团创始人）——科学家出身，战略坚定。但关联交易（华大基因/华大研究院）是投资者核心关切 |
| **Pre-mortem** | **最大死因**：① 持续亏损耗尽现金，被迫融资稀释；② Illumina 大幅降价应对竞争 → MGI 毛利率从 50%+ 降至 30%。概率：中等（25-35%） |

**护城河评分：★★★★（3 年前为 2 星，5 年后可能为 4.5 星——在变宽）**

---

### 2.3 Pacific Biosciences (PacBio) — 长读长之王

| 维度 | 评估 |
|------|------|
| **商业模式** | Revio 测序仪 + 试剂耗材。聚焦长读长细分市场（HiFi reads 准确率 >99.9%） |
| **护城河** | ★★★★（窄而深）。SMRT 技术独有，HiFi reads 在结构变异检测、全长转录组、甲基化等领域无可替代。但：市场总量有限（长读长仅占 NGS ~15%） |
| **竞争对手** | ONT 在便携性上竞争；MGI CycloneSEQ 是新威胁 |
| **行业周期** | 加速期——长读长是 NGS 增速最快的子赛道 |
| **催化剂** | Revio 平台推动成本下降（每 Gb 成本接近短读长）；临床级长读长应用（罕见病/肿瘤） |
| **估值** | PS ~3x（2025 营收 ~$2 亿），持续亏损。市值 ~$8 亿 |
| **管理层** | Christian Henry（CEO）——行业资深，但公司长期亏损消耗耐心 |
| **Pre-mortem** | 最大死因：ONT 和 MGI CycloneSEQ 在长读长市场形成双面包夹，PacBio 市场份额被挤压至 30% 以下。概率：30-40% |

**护城河评分：★★★☆（技术壁垒深但市场窄，新进入者威胁在增加）**

---

### 2.4 Oxford Nanopore Technologies (ONT) — 便携实时之王

| 维度 | 评估 |
|------|------|
| **商业模式** | 纳米孔测序仪（MinION/PromethION）+ 试剂耗材。设备极致小型化（USB 大小） |
| **护城河** | ★★★★☆（独特且深）。纳米孔技术独有——实时测序、超长读长（>4Mb）、设备便携。野外/太空/疫情现场等场景无可替代 |
| **竞争对手** | PacBio（长读长质量之争）、MGI CycloneSEQ（成本优势） |
| **行业周期** | 加速期——实时测序场景在扩展（病原体监测、环境 DNA、农业） |
| **催化剂** | PromethION 平台通量提升；甲基化直接检测（无需 bisulfite 转化） |
| **估值** | PS ~12x，市值 ~£25 亿。市场给了「平台型公司」溢价 |
| **管理层** | Gordon Sanghera（CEO）——科学家+企业家，战略清晰 |
| **Pre-mortem** | 最大死因：准确率始终无法追平 PacBio HiFi → 被限制在利基市场。概率：20-30% |

**护城河评分：★★★★☆（技术独有性最高，但准确率仍逊于 PacBio）**

---

### 2.5 Thermo Fisher Scientific (TMO) — 综合巨头的测序棋子

| 维度 | 评估 |
|------|------|
| **商业模式** | Ion Torrent 测序平台是其庞大科学仪器帝国的极小一部分（TMO 总营收 $400 亿+，测序仅占 ~3%） |
| **护城河** | ★★★☆（在变窄）。Ion Torrent 技术曾经领先，但已被 Illumina 和 MGI 大幅超越。TMO 的测序业务更多是靠捆绑销售（试剂+耗材+仪器）维持 |
| **竞争对手** | 在测序领域不是主要竞争者——TMO 的战略重心不在测序 |
| **估值** | TMO 整体 PE ~25x，不单独评估测序业务 |
| **Pre-mortem** | Ion Torrent 可能在 5 年内被边缘化，TMO 可能退出测序仪市场转而专注于试剂/耗材/服务 |

**护城河评分：★★★（TMO 的护城河在其科学仪器帝国的整体，不在测序仪）**

## 五、投资结论

### 5.1 最优标的：MGI/华大智造 (688114)

**核心理由**：华大智造是当前全球基因测序行业中唯一同时满足以下三个条件的公司：

1. **护城河在变宽**——全读长平台是全球唯一，DNBSEQ 专利自主，中国市场份额碾压
2. **渗透率在甜蜜区**——中国基因测序渗透率远低于美国，装机量年增 28%
3. **估值有安全边际**——PS ~6x，市场给的是「亏损折价」而非「颠覆者溢价」

| 买入逻辑 | 退出条件 |
|---------|---------|
| 全读长平台差异化被市场验证 | 连续 2 季度装机量增速 <15% |
| 2026 年净利润转正 | Illumina 大规模降价导致毛利率降至 40% 以下 |
| 美国市场恢复增长（>20% YoY） | 生物安全法案升级，完全禁止 MGI 在美国销售 |

### 5.2 次优标的：ONT（牛津纳米孔）

**核心理由**：技术独有性最高，但估值偏贵（PS 12x）。适合作为基因测序赛道的「卫星仓位」。

### 5.3 Illumina：持有而非买入

护城河在变窄是最危险的信号。但如果 Illumina 能稳住份额（通过 NovaSeq X 降价和生态锁定），当前 PE 35x 不算贵。**不建议主动买入**，但如果已经持有可以继续持有观察——「王者衰落」可能比想象得慢。

### 5.4 当前持仓关联

| 股票池现有标的 | 与基因测序关系 |
|-------------|------|
| **迈瑞医疗**（器械龙头） | 基因测序不直接关联，但同在医疗器械大赛道 |
| **联影医疗**（影像国产替代） | 与 MGI 同属「国产高端医疗设备替代」逻辑 |
| **Illumina（ILMN）** | 美股黑马池中暂未纳入——建议纳入观察 |
