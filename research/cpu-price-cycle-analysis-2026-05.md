> **资本支出强度是当前 CPU 行业的核心矛盾**。英特尔18A + 台积电N2/N2P + AMD 依赖TSMC产能——三家都在以"不惜代价"的力度投资先进产能。

## 三、护城河分析 — AI 时代 CPU 护城河在变宽还是变窄？

### 3.1 英特尔的护城河

| 维度 | 评级(1-10) | 变化方向 | 说明 |
|------|:---:|:---:|------|
| **x86生态锁定** | 9 | → | 30年+软件生态，企业IT惯性极强 |
| **IDM制造能力** | 7 | ↑ | 18A爬坡中，若成功将重建制造领先 |
| **企业渠道** | 9 | → | Dell/HPE/Lenovo深度绑定，切换成本高 |
| **数据中心份额** | 9 | ↓ | 从98% → 70%+，正在让渡给AMD |
| **AI CPU布局** | 6 | ↑ | Granite Rapids + Diamond Rapids路线图 |

**护城河趋势**：整体在**变窄但未崩塌**。AMD 侵蚀数据中心份额的速度在放缓（EPYC Turin 之后代际提升边际递减），而英特尔的 IDM 2.0 若能成功，反而可能重建护城河。

### 3.2 AMD 的护城河

| 维度 | 评级(1-10) | 变化方向 | 说明 |
|------|:---:|:---:|------|
| **chiplet架构领先** | 8 | → | 被Intel跟进（Meteor Lake），但仍有先发优势 |
| **台积电制程优先** | 9 | ↑ | 台积电量产节奏稳定，代工优势持续 |
| **EPYC性价比** | 9 | → | 同价格核数多，总拥有成本(TCO)优势 |
| **GPU+CPU整合** | 7 | ↑ | MI400 + EPYC 协同，数据中心平台化 |
| **消费市场** | 6 | → | 与Intel交替领先，没有决定性优势 |

**护城河趋势**：**变宽**，但上限有限。AMD 的护城河本质上是"TSMC 制造优势 + chiplet 设计创新"的组合，不拥有任何独有资产。EPYC 份额从 1%→30% 的十年大趋势还在继续，但到 40-50% 后增速会自然放缓。

### 3.3 台积电 — 真正的护城河持有者

CPU 双雄（Intel + AMD）的竞争掩盖了一个核心事实：**台积电才是 CPU 产业链真正的"定价者"**。

- Intel 想用自己的工厂和 18A 摆脱 TSMC 的定价体系
- AMD 则完全绑定了 TSMC 的制程节奏
- 不管谁赢，TSMC 都在涨价——3nm $20,000+/片 → 2nm $25,000+/片

## 四、竞争对手与行业格局 — CPU 定价权的攻防战

### 4.1 三股力量的博弈矩阵

```
                    英特尔 (IDM 2.0)
                    ┌──────────────────┐
                    │ 优势：x86生态    │
                    │ 劣势：制造落后   │
                    │ 策略：降价保份额 │ ← 短期压制CPU涨价
                    └────────┬─────────┘
                             │
            ┌────────────────┼────────────────┐
            │                │                │
            ▼                ▼                ▼
     ┌──────────────┐ ┌──────────────┐ ┌──────────────┐
     │ CSP自研ARM   │ │ AMD EPYC     │ │ 中国CPU替代  │
     │ Graviton等   │ │ 份额爬升中   │ │ 鲲鹏/飞腾等  │
     │              │ │              │ │              │
     │ 威胁等级:★★★ │ │ 威胁等级:★★★★│ │ 威胁等级:★★  │
     └──────────────┘ └──────────────┘ └──────────────┘
```

### 4.2 竞争格局的核心矛盾

**AMD 不想涨价**：AMD 正在用"同价多核"策略从 Intel 手中抢份额。涨价会削弱这个优势。

**Intel 需要涨价**：Intel 在 18A 上烧了几百亿美元，晶圆厂固定成本极高。只有 ASP 上去了，IDM 2.0 才能跑通。

**CSP 不想任何人涨价**：AWS、Google、微软都已部署自研 ARM CPU（Graviton / Axion / Cobalt），不是为了性能，是为了**定价权底线**——自研 CPU 的存在本身就是对 x86 涨价的威慑。

**台积电微笑旁观**：不管谁卖 CPU，都要在台积电流片。3nm→2nm→A14 的 ASP 阶梯上涨是不可逆的。

### 4.3 关键竞争动态

| 事件 | 影响力 | 对 CPU 价格影响 |
|------|:---:|:---:|
| Intel 18A 2026H2 量产 | ★★★★★ | 若成功→缓解供给瓶颈→降价压力 |
| AMD Zen 6 / EPYC Venice (2027) | ★★★★ | 延续性能优势→Intel被迫降价 |
| AWS Graviton4 大规模部署 | ★★★ | ARM替代加速→x86定价权削弱 |
| 中国信创CPU采购（鲲鹏/飞腾/海光） | ★★★ | 国内CPU价格有独立定价体系 |
| Diamond Rapids 2027年发布 | ★★★★ | Intel高端供给释放→缓解服务器端涨价 |

## 五、行业周期定位 — 我们现在在哪里？

### 5.1 半导体周期时钟

```
              ┌─────────────┐
              │  过热期      │
              │  价格飙升    │ ← 存储/NAND 在这里 (2026Q1-Q2)
              │  CapEx 高峰  │
              └──────┬───────┘
                     │
    ┌────────────────┼────────────────┐
    │                │                │
    ▼                ▼                ▼
┌────────┐    ┌──────────┐    ┌──────────┐
│ 复苏期 │ ← │ 扩张期    │ ← │ CPU 在这里│
│        │   │          │   │ (中段偏早) │
└────────┘   └──────────┘   └──────────┘
     ▲                            │
     │                            │
     └────────────────────────────┘
           下行期 (2027H2?)
```

**CPU 周期定位与存储不同**：

- **存储（DRAM/NAND）处于过热期**：价格Q1环比+83-95%，三星营业利润率 43%，产能利用率接近极限。历史上存储周期从过热到崩盘只需 6-12 个月。
- **CPU处于扩张期中段偏早**：AI CPU 需求刚刚开始大规模出现（Agent推理、CSP自建数据中心），供给瓶颈比存储更严重（先进制程产能扩张需要 3-5 年），且 CPU 的定价结构比存储更"粘性"（企业 IT 预算锁定、长期合同）。

### 5.2 关键周期指标

| 指标 | 当前值 | 周期位置判断 | 历史参考 |
|------|------|:---:|------|
| TSMC CapEx / 营收比 | 35-38% | 高位 | 2021年曾达50%+ |
| 英特尔 CapEx / 营收比 | 40%+ | 极高 | 18A战略投资，非正常周期 |
| 全球硅晶圆出货量增速 | +13% YoY | 扩张期 | 2021年曾+14% |
| 晶圆厂设备支出增速 | +11-18% | 扩张期 | 2024年曾负增长 |
| 先进制程产能利用率 | >95% | 供不应求 | — |
| 成熟制程产能利用率 | 75-85% | 结构性分化 | — |

**核心判断**：CPU 的涨价周期**比存储滞后约 6-9 个月**。存储价格已经在过热区域，CPU 还在爬坡期。这意味着 CPU 的涨价趋势有更长的跑道——但也意味着存储崩盘时 CPU 不太可能独善其身。

### 6.3 AI 需求的三级火箭

```
第一级（已点燃）    AI训练 GPU需求         ← 大规模算力集中采购
第二级（正在点火）  AI推理 CPU+GPU混合      ← Agent型工作负载天然适配CPU
第三级（待点燃）    AI Agent 自主执行        ← 每个Agent需要持续的CPU资源
```

**第二级火箭是关键**。AI 推理从 GPU 向 CPU 迁移不是"如果"，而是"多快"。原因：
- Agent 型 AI 需要多步推理、工具调用、状态管理——这些天然是 CPU 擅长的工作
- GPU 在推理批处理中有吞吐量优势，但 CPU 在单请求延迟和成本上更优
- 行业内已经开始讨论"CPU:GPU 配比从 1:8 向 1:1 甚至 CPU 更多倾斜"

## 七、估值分析 — CPU 产业链核心标的

### 7.1 英特尔 (INTC) — 冤大头定价权

| 指标 | 数据 |
|------|------|
| 股价 (2026.5.5) | ~$30-35（Q1财报后+28%） |
| 市值 | ~$1,200-1,400亿 |
| TTM PE | 亏损/微利，PE无意义 |
| FY2026E 营收 | ~$560-580亿 |
| 毛利率 | ~45%（目标2027年回到55%+） |

**英特尔是这轮CPU涨价的"冤大头"**——拥有最大产能，但制造落后；需求飙升受益，但成本结构拖累利润。市场给英特尔的定价中有大量"18A成功与否"的博弈。

**若18A成功**：英特尔从"制造落后方"变成"先进制造方"，毛利率从45% → 55%+，PE重建 → 股价重估。

**若18A失败**：IDM 2.0解体，分拆制造业务，CPU ASP上不去 → 股价继续低迷。

| 情景 | 目标价 | vs 现价 ~$33 |
|------|:---:|:---:|
| 乐观（18A成功 + AI CPU需求旺盛） | $55-65 | +70-100% |
| 基准（18A部分成功，毛利率缓慢改善） | $35-45 | +10-40% |
| 悲观（18A失败，份额继续流失） | $15-20 | -40-50% |

**赔率**：1:3（下行40% vs 上行100%）。这是一笔赔率还行的逆向投资——但前提是你相信18A能成。

### 7.2 AMD (AMD) — 最直接受益于涨价

| 指标 | 数据 |

| 市值 | ~$1,800-2,000亿 |
| TTM PE | ~25-30x |

AMD 是 CPU 涨价周期中**最纯粹的受益者**：
- 不承担 IDM 的重资产风险

- 数据中心 EPYC 份额仍有爬升空间（当前 ~30%，中长期目标 40-50%）

**估值逻辑**：AMD 现有估值已 partially 反映 EPYC 份额爬升叙事，但没有完全反映"AI CPU 需求结构性增长"的潜在价值。如果 CPU:GPU 配比确实向 1:1 收敛，AMD 的成长天花板将被显著抬高。

## 八、管理层与治理 — 谁在为涨价做正确的事？

### 8.1 英特尔 — 新管理层在正确但漫长的路上

- **CEO Lip-Bu Tan（2025.3上任）**：华裔芯片老将，Cadence前CEO，技术+商业双重背景
- **策略转变**：从"什么都做"转向"核心是制造"，大幅裁员+砍非核心业务
- **资本配置**：$200亿+ CapEx投入18A，削减分红，出售Altera等非核心资产
- **正面信号**：Q1 2026财报超预期，市场对新管理层开始给予信任（股价+28%）
- **负面信号**：18A是"掷骰子"级别的赌注——成则王者归来，败则帝国解体
- **对涨价的态度**：英特尔**必须涨价**——没有高ASP就无法支撑IDM 2.0的固定成本。管理层对此心知肚明。

### 8.2 AMD — Lisa Su 的十年

- **CEO Lisa Su（2014年至今）**：半导体行业最佳CEO之一
- **执行力**：Zen微架构规划到2027年（Zen 6 Venice），路线图历史上最稳定承诺
- **资本配置**：回购~$50亿/年，适时收购（Xilinx/Pensando），谨慎扩张
- **风险**：Lisa Su 时代已进入第12年，任何继任不确定性都会引发估值担忧
- **对涨价的态度**：AMD **不想带头涨价**。AMD 的竞争力来自"同价多核"的TCO优势。如果整个行业都在涨价，AMD会跟涨，但不会主动引领。

## 九、逆向验证（Pre-mortem） — 假设 3 年后 CPU 投资腰斩

### 9.1 死因一：AI 工作负载重新 GPU 化

**场景**：GPU 推理成本再降 70%（Blackwell → Rubin → Rubin Ultra 的路线图），而 CPU 推理性价比优势被蚕食。加上 NVIDIA 通过 CUDA-X 推理库将软件生态优势延伸到推理侧，Agent 工作负载最终仍锁定在 GPU 上。

**结果**：CPU:GPU 配比的"1:1"叙事破产，CPU 重新回到被 GPU 挤压的结构性下行通道。

**认真审视**：过去 10 年 GPU 一直在"吃掉"CPU 的计算负载（从图形到AI训练到推理），NVIDIA 有极强的动机和资源去阻止 CPU 夺回失地。**这不是杞人忧天**。

### 9.2 死因二：英特尔 18A 大获成功 → 供给过剩

**场景**：18A 良率超预期，2027 年英特尔释放大量先进制程 CPU 产能。同时台积电 N2 量产，三星 2nm GAA 也加入竞争。三大制造源同时放量 → CPU 供给从瓶颈变为过剩 → ASP 不涨反跌。

**结果**：涨价叙事被自己的供给侧逻辑反杀。

**认真审视**：半导体行业历史上每次"产能紧张→暴涨→大规模CapEx→过剩→崩盘"周期无一例外。本轮 CPU 涨价也不会例外。问题是：**拐点在 2027H2 还是 2028H2？**

### 9.3 死因三：CSP 自研 ARM CPU 大规模替代 x86

**场景**：AWS Graviton5 + Google Axion v2 + 微软 Cobalt v2 在 2027-2028 年合计占据 CSP 工作负载 30%+，x86 的"独家"地位被打破。英特尔的 ASP 不仅涨不上去，还要降价来与自研 ARM 竞争。

**结果**：x86 CPU 定价权永久性削弱。

**认真审视**：这与 NVIDIA 面临的 CSP 自研 ASIC 威胁是同一逻辑——**高度集中的客户拥有强烈的自研动机和足够的资金能力**。CPU 面临比 GPU 更大的风险，因为 ARM 授权模式降低了自研 CPU 的门槛（买 ARM Neoverse IP 比自己设计 GPU 容易得多）。

### 9.4 最可能死因排序

| 排名 | 死因 | 3年内概率 | 腰斩概率 |
|:---:|------|:---:|:---:|
| 1 | 半导体周期自然见顶（供给过剩） | 35% | ★★★★ |
| 2 | AI工作负载GPU化，CPU需求回落 | 20% | ★★★★★ |
| 3 | CSP自研ARM替代x86 | 15% | ★★★ |
| 4 | 地缘风险（台海/关税） | 10% | ★★★★★ |

### 9.5 什么信号会让我改变判断？

**离场信号**：
1. CSP 年度 CapEx 指引增速从 +30% 骤降至 +5%（AI 投资退潮）
2. DRAM 合约价连续两个季度 QoQ 转负（存储崩盘→半导体情绪逆转）
3. 英特尔 18A 良率达到 80%+ 并宣布大规模量产（供给冲击）
4. AWS 或 Google 宣布自研 CPU 在 50%+ 工作负载部署（x86 替代加速）

**加仓信号**：
1. AI Agent 应用出现"ChatGPT 时刻"级别的爆款（Agent 推理 CPU 需求加速）
2. 英特尔 18A 再次延迟（供给瓶颈延长）
3. 台积电再次上调 2nm wafer 价格（成本推动的 ASP 上涨）

加入 CPU 标的后，这个情景的风险**没有分散，而是在叠加**——英特尔和 AMD 在同一情景下可能跌 30-50%。
