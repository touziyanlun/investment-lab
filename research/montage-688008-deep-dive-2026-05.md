# 澜起科技（688008.SH / 06809.HK）深度研究报告

> **分析日期**：2026-05-05
> **投资期限**：5年以上
> **核心结论**：全球内存接口芯片绝对龙头，JEDEC唯一中国董事会成员构筑不可替代的标准壁垒。DDR5渗透率提升+PCIe Retimer放量+CXL MXC储备，三条增长曲线接力清晰。A+H双平台打开全球定价权重估空间。当前估值合理偏低，是AI产业链中确定性较高的核心标的。

## 一、公司概况

澜起科技（Montage Technology），成立于2004年，2019年科创板首批挂牌（688008），2026年2月港股上市（06809.HK），实现A+H双平台布局。核心业务是**互连芯片（Interconnect Chips）**——不直接参与计算，却决定了CPU、GPU、内存、存储之间的高速数据流动效率，是AI数据中心的关键基础器件供应商。

## 三、护城河 — 为什么对手抢不走生意？

### 3.1 核心壁垒：JEDEC标准制定权 ⭐⭐⭐⭐⭐

**这是澜起最深的护城河，是所有估值溢价的根本来源。**

| 要点 | 详情 |
|------|------|
| 身份 | **JEDEC董事会唯一中国芯片企业** |
| DDR4时代 | 发明"1+9"分布式缓冲内存子系统框架，被采纳为国际标准 |
| DDR5时代 | 牵头制定RCD、MDB、CKD芯片国际标准 |
| 技术代差 | 领先竞争对手（瑞萨、Rambus）**6-12个月** |
| 物理壁垒 | **每条DDR5服务器内存必须内置RCD芯片，无替代方案** |
| 专利储备 | 全球1,200+项专利 |

> 为什么重要？因为AI服务器升级内存的唯一路径就是DDR5，而DDR5的唯一路径就是RCD芯片。澜起在DDR5高端市场份额达40-50%，新子代切换初期定价权最强——这构成了一道竞争对手几乎无法越过的"标准制定权+物理必要性+定价权"三重壁垒。

**护城河趋势**：↑ 变宽中。DDR5渗透率从~60%向~90%提升，子代从第四子代向第六子代迭代，ASP持续提升。PCIe Retimer和CXL MXC两条新增长曲线进一步加深技术储备壁垒。

### 3.2 第二壁垒：PCIe Retimer国产突破 ⭐⭐⭐⭐

- 2024年出货量突破百万颗，打破美国Astera Labs的长期垄断
- 2026年1月发布PCIe 6.x/CXL 3.x AEC解决方案，采用自研SerDes技术
- 全球唯二可提供DDR5第一子代MRCD/MDB芯片的供应商

### 3.3 第三壁垒：客户粘性+认证周期 ⭐⭐⭐⭐

- 下游客户：三星、SK海力士、美光、英特尔、浪潮、新华三
- 内存接口芯片需要与CPU平台（英特尔/AMD）深度适配认证，认证周期12-18个月
- 一旦进入供应链，除非出现重大质量或断供问题，极难被替换

## 八、Pre-mortem 逆向验证 — 假设3年后腰斩甚至归零

### 最可能的死因 #1：AI资本开支周期见顶（概率：25%）

**情景**：2027年全球CSP同时削减AI基础设施投入（产能过剩+回报不及预期），服务器出货量下降30%。DDR5需求锐减+Retimer订单骤降，澜起营收出现负增长。高固定研发成本导致利润加速恶化，PE被杀至25-30倍，股价跌至**80-100元**（-40%至-50%）。

**反驳**：DDR5渗透率提升是结构性趋势（非周期性），即使服务器出货下降，DDR5对DDR4的替代仍会持续。且澜起负债极低、净现金充裕，抗周期能力强。

### 最可能的死因 #2：Astera Labs价格战+客户流失（概率：15%）

**情景**：ALAB将中低端Retimer价格下调40%，澜起被迫跟进——Retimer毛利率从60%+降至30%。同时三星/海力士开始扶持瑞萨作为第二供应商，澜起在内存接口芯片的份额从37%降至30%以下。

**反驳**：内存接口芯片的验证周期长达12-18个月，客户切换成本极高。DDR5新子代产品澜起领先6-12个月，客户不会因为短期价格放弃技术领先的供应商。

### 结论

澜起**归零的路径几乎不存在**——它的芯片是AI服务器的物理必需品，需求只会因周期波动而非消失。最差情景是股价腰斩，但这将对应PE 25-30倍买入全球龙头——仍然是好公司的好价格。
