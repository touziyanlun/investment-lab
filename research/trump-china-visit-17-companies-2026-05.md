# 特朗普2026年5月访华随行企业深度分析

> 2026年5月18日 | 专题研究 | 标签：#特朗普访华 #中美关系 #企业深度分析  
> 17家随行企业 • 逐家深度拆解 • 投资含义

---

## 背景

2026年5月13-15日，美国总统特朗普对北京进行国事访问，与中国国家主席习近平就贸易、科技、AI、半导体出口、台湾、芬太尼及伊朗战争经济影响等议题举行峰会。

白宫邀请了 **17位美国企业CEO/高管** 随行，覆盖科技、半导体、金融、航空航天、农业等领域。这是自2017年11月特朗普首次访华以来最大规模的美国企业访华代表团。

---

## 一、科技与半导体（6家）

### 1. 苹果 Apple (AAPL) — Tim Cook

| 维度 | 内容 |
|------|------|
| **市值** | ~4.4万亿美元 |
| **中国营收** | 大中华区 ~17-19%，FY2026 Q1 达 255亿美元（+38%） |
| **供应链依赖** | ~80% iPhone 在中国组装（富士康/和硕），供应商网络深度嵌入中国 |
| **核心诉求** | 确保 iPhone 关税豁免延续；为 CEO 交接铺路（Tim Cook 9月卸任→John Ternus 接任） |
| **访华成果** | Cook 人脉深厚，关税豁免大概率延续。iPhone 17 在中国表现强劲（Q2 iPhone 营收 +21.7%） |
| **主要风险** | 华为（24%中国市场份额）和小米竞争；关税重新征收风险；领导层断层 |
| **投资判断** | Q2营收1112亿美元（+17%），EPS $2.01。但 AI 落后（股价 YTD -5%），CEO 更替不确定性大 |

### 2. 英伟达 Nvidia (NVDA) — 黄仁勋 🏆

| 维度 | 内容 |
|------|------|
| **市值** | ~5.5万亿美元（全球最高） |
| **中国营收** | 几乎为 0（出口管制后）——此前控制中国 ~95% AI 芯片市场 |
| **核心诉求** | 解除 H200 AI 芯片对华出口限制 |
| **访华成果** | **访华期间已获批！** 美国批准向 10 家中国公司销售 H200，潜在解锁 ~260 亿美元营收 |
| **主要风险** | 中国客户基于"买国货"政策可能拒绝采购；出口管制随时可能收紧 |
| **投资判断** | 本次访问最大实质性突破。但全球 AI CAPEX 才是核心驱动力，中国是"锦上添花" |

### 3. 美光 Micron (MU) — Sanjay Mehrotra

| 维度 | 内容 |
|------|------|
| **市值** | ~1500亿美元 |
| **中国营收** | ~12%（2025年约26.4亿美元，2024年为30.5亿，下降趋势） |
| **核心诉求** | 修复关系——2023年被中国以国家安全为由禁售（数据中心市场） |
| **访华成果** | **突破可能性最低**。中国视美光为华盛顿"鹰派"，信任赤字严重 |
| **主要风险** | 长鑫存储/长江存储快速替代；HBM 对华出口受制裁限制 |
| **投资判断** | Q2 营收 238.6亿美元（+196%）由 AI/HBM 驱动，基本面强劲。但中国 12% 营收可能永久性损失 |

### 4. 高通 Qualcomm (QCOM) — Cristiano Amon

| 维度 | 内容 |
|------|------|
| **市值** | ~1700亿美元 |
| **中国营收** | **~46%**（17家企业中最高！）——骁龙处理器驱动绝大多数中国安卓手机 |
| **核心诉求** | 推动 6G 标准合作；稳定中国智能手机市场；汽车芯片业务（+38% YoY） |
| **访华成果** | 中高。中国手机库存 Q3 见底→反转信号明确 |
| **主要风险** | 苹果自研 Modem（2027年全面替代高通）；46% 营收集中度极高 |
| **投资判断** | Q2 营收 106亿美元。中国底部反转进行中，AI 数据中心芯片开辟新业务线。但苹果 Modem 替代是悬顶之剑 |

### 5. Coherent (COHR) — Jim Anderson

| 维度 | 内容 |
|------|------|
| **市值** | ~790亿美元 |
| **中国营收** | 中等（数据中心光学组件——收发器、激光器） |
| **核心诉求** | 明确 AI 数据中心跨境建设政策；保护光学网络组件对华供应 |
| **访华成果** | 作为英伟达供应链关键环节（英伟达投资20亿美元），政策确定性至关重要 |
| **主要风险** | 中国限制光电子元件出口；中美数据中心供应链"脱钩" |
| **投资判断** | Q3 营收 18.1亿美元（+21%）。AI 光学超级周期起点，数据中心营收占 72% |

### 6. Illumina (ILMN) — Jacob Thaysen

| 维度 | 内容 |
|------|------|
| **市值** | ~200亿美元 |
| **中国营收** | ~7%（从峰值 ~5亿美元降至 ~3亿美元，连续6季度下滑） |
| **核心诉求** | 从中国"不可靠实体清单"（UEL）中移除——2025年2月被列入 |
| **访华成果** | 防御性。中国市场已被华大智造主导（63.8%份额，价格低30-50%） |
| **主要风险** | 华大智造竞争 + 中国本土化政策 → 中国营收下滑是结构性而非周期性 |
| **投资判断** | 详见基因测序行业报告。护城河持续变窄，从 UEL 移除仅带来有限上行空间 |

## 二、金融与支付（6家）

### 7. 花旗 Citigroup (C) — Jane Fraser 🏆

| 维度 | 内容 |
|------|------|
| **市值** | ~1500亿美元 |
| **中国营收** | 中等（消费者财富管理、企业与机构银行、市场业务） |
| **核心诉求** | 获取中国全资证券牌照——已苦等4年 |
| **访华成果** | **此行最大赢家！** 中国证监会批准花旗全资证券牌照——成为第7家获此资格的外资机构 |
| **主要风险** | 牌照变现需要数年；2025年裁减中国3500个技术岗位（成本削减） |
| **投资判断** | Q1 营收破十年纪录（246亿美元，+14%），EPS 超预期15%。中国牌照是长期催化剂 |

### 8. 高盛 Goldman Sachs (GS) — David Solomon

| 维度 | 内容 |
|------|------|
| **市值** | ~1700亿美元 |
| **中国营收** | <5%（亚洲除日本整体 ~15-18%） |
| **核心诉求** | 会见央行副行长和外管局，推动基金代销牌照扩展和在岸债券/衍生品准入 |
| **访华成果** | 渐进式进展。中国营收贡献太小难以左右整体业绩 |
| **投资判断** | Q1 净收入 172亿美元（历史第二高）。中国是"增量期权"而非核心驱动力 |

### 9. 贝莱德 BlackRock (BLK) — Larry Fink

| 维度 | 内容 |
|------|------|
| **市值** | ~1680亿美元 |
| **中国营收** | <3-5%（机构委托和 ETF） |
| **核心诉求** | 推动 23亿美元收购 CK Hutchison 港口交易（含巴拿马运河两港口）获中国默许 |
| **访华成果** | 防御性——该交易面临中美在巴拿马运河的战略竞争审查 |
| **投资判断** | Q1 营收 67亿美元（+27%），净流入 1300亿美元创纪录。中国对营收影响小 |

### 10. 黑石 Blackstone (BX) — Stephen Schwarzman

| 维度 | 内容 |
|------|------|
| **市值** | ~960亿美元 |
| **中国营收** | 亚洲整体 ~15-20%（中国占比在下降） |
| **核心诉求** | 外资 PE 募资/退出渠道监管透明度；关注不良房地产和信贷机会 |
| **访华成果** | 低紧迫度——中国在 BX 亚洲战略中的地位正被印度/日本取代 |
| **投资判断** | Q1 营收 36.2亿美元超预期，AUM 1.3万亿美元（+12%）。但股价 YTD -30% |

### 11. Visa (V) — Ryan McInerney

| 维度 | 内容 |
|------|------|
| **市值** | ~5900亿美元 |
| **中国营收** | ~2-3%（仅跨境交易——中国人境外消费、外国卡在华使用） |
| **核心诉求** | 以 **100%持股** 方式进入中国国内银行卡清算市场——前所未有的要求 |
| **访华成果** | 外交性接触，离批准尚需数年。对手美国运通（2020年）/万事达卡（2023年）已获批 |
| **主要风险** | 银联 >90% 份额 + 支付宝/微信支付主导 → 银行卡清算牌照价值存疑 |
| **投资判断** | 中国国内卡市场年交易额 ~17万亿美元，但数字支付正取代实体卡 |

### 12. 万事达卡 Mastercard (MA) — Michael Miebach

| 维度 | 内容 |
|------|------|
| **市值** | ~4600亿美元 |
| **中国营收** | ~2-3%（主要为跨境交易） |
| **核心诉求** | 争取提高网联合资企业持股比例（目前为少数股权） |
| **访华成果** | 已于2023年通过 JV 获批牌照，2024年5月开始处理国内交易——领先 Visa |
| **投资判断** | Q1 营收 84亿美元（+16%）。4月消费放缓导致股价单日 -4.4% |

## 三、工业/航空/EV/农业（5家 + SpaceX）

### 13. 特斯拉 Tesla (TSLA) — Elon Musk

| 维度 | 内容 |
|------|------|
| **市值** | ~9500亿美元 |
| **中国营收** | ~22%（上海超级工厂贡献全球产量 ~52%，年产能 >95万辆） |
| **核心诉求** | 全自动驾驶（FSD）中国审批（推迟至 Q3 2026）；购买29亿美元太阳能设备 |

### 14. 波音 Boeing (BA) — Kelly Ortberg 📝

| **主要风险** | 市场反应冷淡——股价当日下跌 4.7%；产能严重受限（发动机/起落架交期18个月） |
| **投资判断** | 结束十年冰封期意义重大，但现金流需2年以上才能体现 |

## 四、总结：17家企业访华得失矩阵

| 层级 | 企业 | 核心成果 | 后续观察点 |
|:---:|------|------|------|
| 🏆 大赢家 | **花旗、英伟达** | 证券牌照获批 / H200芯片获批 | 花旗牌照变现节奏；中国客户是否下单 H200 |
| 📝 实质性进展 | **波音、高通、苹果** | 200架飞机 / 6G合作 / 关税豁免 | 波音产能执行；中国手机库存见底时点 |
| ⏳ 长期布局 | **Visa、万事达卡、贝莱德、高盛** | 市场准入谈判 / 关系维护 | 中国金融市场开放节奏 |
| 🔒 难以突破 | **美光、Illumina** | 信任赤字大 / 中国本土替代已成定局 | 长鑫/华大智造竞争格局演变 |
| 🎯 其他目的 | **特斯拉、SpaceX、Meta、Coherent** | FSD / IPO造势 / 保护广告收入 | FSD审批进展；SpaceX IPO 定价 |
| 🌾 贸易相关 | **嘉吉、GE Aerospace、黑石** | 农产品贸易 / 发动机供应 / PE准入 | 中国实际采购量；波音交付时间线 |

### 关键判断

1. **"雷声大雨点小"**（CNN评价）：多数企业获得的是口头承诺而非书面协议
2. **英伟达 H200 获批是最大实质性突破**，但中国客户是否实际下单仍存不确定性
3. **花旗证券牌照**是金融领域的最大赢家——从申请到获批跨越5年
4. **美光和 Illumina** 的困境是结构性的——中国本土替代已不可逆转
5. **未获邀请的关键缺席者**：OpenAI（Sam Altman 因官司未能参加）、微软、亚马逊、谷歌
