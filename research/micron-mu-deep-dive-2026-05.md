## 三、估值 — 本轮最便宜的AI标的？

| 指标 | 数据 |
|------|------|
| 股价 | ~$450-530 |
| TTM PE | ~23.6x |
| **Forward PE(FY2026)** | **~7-8x** |
| 半导体行业均值 | ~27x |
| 分析师目标价 | $543(均值)/$1,000(DA Davidson最高) |

**核心观点**：Forward PE 7-8x vs 行业均值27x = **深度折价**。如果AI驱动的存储需求确实结构性改变了行业(不再3年一周期)，这将是未来12个月最大的估值修复机会之一。

**反方观点**：历史上MU Forward PE 7-8x恰恰是周期顶部的特征(PE = EPS飙升/股价提前反映)。这一次不一样吗？

## 四、风险

1. **76-81%毛利率不可持续**：存储器历史上毛利率峰值后总是暴跌
2. **三星$730亿CapEx**可能在2027-2028年引发供给过剩
3. **"Earnings Paradox"**：Q2财报beat却跌6%——预期已极度高企

---

## 五、Pre-mortem

**最可能死因**：2027年存储供给过剩 → DRAM价格从$5跌至$2 → EPS从$60跌至$15 → PE从8x膨胀至30x(TTM) → 股价跌回$200。这在2018-2019、2015-2016、2011-2012三次发生过。

**这次真的不一样吗？** 5年长协+AI需求确实是新变量，但存储行业80年历史的周期惯性不可低估。

## 七、投资结论

| 维度 | 评分 |
|------|:---:|
| 商业模式 | 8/10 |
| 护城河 | 8/10 |
| 估值 | **10/10**(Forward PE 7-8x) |
| 风险调整后赔率 | 7/10 |
| **综合** | **8.25/10** |

**建议仓位：6-8%**。这是9家中赔率最高的标的——如果存储行业真的"这次不一样"，Forward PE从8x修复至15x = 翻倍空间；如果周期回归，7-8x的Forward PE提供了部分下行保护(盈利下跌但PE扩张)。

**芒格会说**："以合理的价格买伟大的公司"——美光不是"伟大"(存储终究是商品)，但当前价格**极其合理**。
